## When to Use

Activate when the user is **drafting a consulting or service proposal** for:
- UE5 optimization services (ArchViz firms)
- VR/AR interactive solution development
- 3D visualization and rendering services
- BIM family creation or integration
- Digital twin development
- Training/workshop services (ArchViz, VR, game dev)

## NOT for
- Job applications or resumes (use general writing)
- Legal contracts (see a lawyer)
- Market research reports (use `consulting-analysis`)

## Proposal Structure Template

### 1. Executive Summary
```
CONSULTANCY PROPOSAL
[Service Type]
Prepared by: [Your Name / Studio Name]
Date: [Date]
Client: [Client Name / Firm]

We propose to [core service offering in one line].
This engagement will [key outcome — e.g., improve FPS, create interactive experience].
```

### 2. Client Situation & Pain Points
List 2-3 specific pain points the client is experiencing:
- *"Your ArchViz walkthroughs run at 15-20 FPS, stuttering during client presentations."*
- *"You're losing deals because competitors offer interactive VR walkthroughs and you don't."*

### 3. Proposed Solution

**Scope of Work** (bullet points, ~5-8 items):
```
- Performance audit of current UE5 project (draw calls, LODs, Nanite/Lumen settings)
- Asset optimization: LOD generation, texture budget, mesh merging
- Rendering pipeline tuning for ArchViz output
- VR implementation for client walkthroughs
- Documentation + training for your team
```

### 4. Deliverables
```
Week 1: Performance audit report + optimization roadmap
Week 2: Asset optimization pass (LODs, textures, meshes)
Week 3: Pipeline tuning + quality validation
Week 4: Final delivery + team handoff
```

### 5. Pricing Models

| Model | Best For | Structure |
|---|---|---|
| Fixed Price | Defined scope, clear deliverables | ~₹X per phase, paid 50% upfront |
| Hourly Consulting | Ongoing optimization, undefined scope | ₹X/hr, billed weekly |
| Retainer | Monthly optimization/maintenance | ₹X/month, X hours included |
| Commission | Revenue-share on VR projects | % of project value |

**India Market Rates (2025-2026):**
- UE5 optimization consulting: ₹15,000-₹30,000/day (₹2,000-₹4,000/hr)
- VR/AR development: ₹50,000-₹2,00,000 per project (depending on scope)
- 3D visualization: ₹5,000-₹25,000 per render
